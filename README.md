# Cydial
## Cydial Connect via Bluetooth App

### Prerequisites
* Arduino UNO
* MG996R Servo
* Some Jumper Cables, M-M, M-F, F-F
* AA Battery X4
* AA 5V plugin
* Keyestudio 16x12 bit PWM Shield
* Breadboard
* Arduino-USB Cable
* HC-06 (ZS-040) Bluetooth
* LCD
* Computer with Arduino IDE
* Android Smartphone/Tablet
